# 📸 Carômetro – Sistema de Cadastro de Estudantes

O **Carômetro** é um sistema desenvolvido com fins educacionais pelo **SENAC**, com o objetivo de realizar o cadastro e gerenciamento de estudantes de forma simples, visual e intuitiva.

A aplicação permite registrar informações básicas dos alunos, adicionar fotos e gerar documentos, facilitando a organização e identificação dos estudantes em ambientes acadêmicos.

---

## 🧩 Funcionalidades

* 🔎 **Busca de estudantes**

  * Campo de pesquisa para localizar alunos cadastrados rapidamente.

* 📝 **Cadastro de estudante**

  * Inserção do nome do estudante.
  * Registro simples e direto.

* 🖼️ **Upload de foto**

  * Permite adicionar uma imagem para identificação visual do aluno.

* 📄 **Geração de PDF**

  * Exporta os dados cadastrados em formato PDF.

* ✏️ **Edição e exclusão**

  * Alterar ou remover registros existentes.
